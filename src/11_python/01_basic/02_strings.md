# Strings #
* string
```python
string = 'String' 
```
* multiline string
```python
string = '''
String in first line
String in second line
''' 
```

* string index
```python
string = 'String'
string[0]
``` 
* negative index
```python
string = 'String'
string[-1] // g
```

* substring
```python
string = 'String'
string[0:3] //Str
string[:3] //Str
string[2:] //ring
string[:] //String
string[1:-1] //trin
``` 

* concatenate strings
```python
str1 = 'String1'
str2 = 'String2'

msg = 'string1 = {str1} and string2 = {str2}'
```
