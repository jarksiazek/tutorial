# Variables #
* declare variable "price"
```python
price = 10 
print(price) //10
```
* change "price" value to 20
```python
price = 10 
price = 20 
print(price) //20
```

* other types of variables
```python
price = 10 
decimal = 2.1 
name = 'Jarek'
boolean_variable = True
```

* type conversion - int(), float()
```python
number_string = '12'
number_int = int(12)
```

* show type - type(), float()
```python
number_string = '12'
type = type(12)
```
