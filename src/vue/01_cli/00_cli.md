# CLI #
### Links ###
https://cli.vuejs.org

### CLI commends ###
* global installation 
```bash
npm install -g vue-cli
```
* creating project - go to direction  
```bash
vue init webpack-simple vue-cli
cd vue-cli
npm install
npm run dev
```
