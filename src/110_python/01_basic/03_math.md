# Mathematic #
### Mathematic operations ### 
* // return integer
```python
print(10 / 3) # 3.33333335
print(10 // 3) # 3.33333335
```

* exponentiation
```python
print(10 ** 3) # 1000
```

### Mathematic functions ### 
* round
```python
round(2.9)
```
* absolute
```python
abs(2.9)
```

* import math module to have access to more functions
https://docs.python.org/3/library/math.html
```python
import math

math.ceil(2.9)
```
